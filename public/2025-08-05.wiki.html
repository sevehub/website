Creating a YouTube Short with FFmpeg: Showcasing My Favorite Stationery

As a stationery enthusiast, I have a collection of office supplies and other delightful items that I want to showcase in a YouTube Short. This blog post outlines how I created the simple video below, using only Typst and FFmpeg. The process was straightforward and fun, and I believe you will find it beneficial for your upcoming projects.

Why Use FFmpeg?

FFmpeg is a powerful cli tool that allows me to edit and convert my videos effortlessly. Whether I want to add some background music, or ensure my videos the right format for YouTube Shorts, FFmpeg is my go-to digital toolbox.

We have already shown how to install FFmpeg on Windows using the Mingw64 subsystem. If you haven't set it up yet, just follow those steps, and you'll be ready to go.

Installing FFmpeg on a Linux system involves using the package manager specific to your distribution. Each Linux distribution has its own package management system, which means the commands and methods for installation can vary.

Similar to MSYS2/Mingw64, Arch Linux utilizes the Pacman package manager, known for its simplicity and speed. You can install FFmpeg from the cli with:

   sudo pacman -S ffmpeg

Creating a Video with Typst/FFmpeg

I began by creating a document in Typst, where each page represents a frame in the final video. This approach enabled me to design each frame with the specific visuals I wanted to showcase.

Once the document was ready, I utilized the Vim plugin, typstpowershell, to export it as a series of PNG images using the command :TypstCompileToPNG. Each image corresponds to a page in the compiled PDF output, making it easy to manage the frames for this video.

Next, I needed to create a file list for FFmpeg. This list is essential because it specifies the images I want to include in the video and allows me to set the duration for each frame.

For this example I used the text file below (stationery.ffmpeg):

    file 'stationery_1of6.png'
    duration 5
    file 'stationery_2of6.png'
    duration 5
    file 'stationery_3of6.png'
    duration 5
    file 'stationery_4of6.png'
    duration 5
    file 'stationery_5of6.png'
    duration 5
    file 'stationery_6of6.png'
    duration 10

I can then generate an intermediate video without background music using the following FFmpeg command line:

   ffmpeg -f concat -safe 0 -i stationery.ffmpeg -c:v libx264 -pix_fmt yuv420p output.mp4 -report

To make this example even more engaging, I also added some background music. The following command combines the previous video with a music file I downloaded from Pixabay.

   ffmpeg -i output.mp4 -i cyberpunk.mp3 -c:v copy -c:a aac -strict experimental -shortest stationery.mp4
