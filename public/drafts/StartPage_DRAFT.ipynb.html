Start Page¶

Mozilla Thunderbird Configuration¶

Set the Start Page¶

Tools → Options → General

Copy/Paste onto Location

Add-ons¶

Quicktext

Install Quicktext from the Add-ons browser.

Tools → Quicktext → General

Copy/Paste https://tessarinseve.pythonanywhere.com/staticweb/quicktextT.xml onto Import on startup

Key Bindings¶

Install TBKEYS-LITE from the Add-ons browser.

Tools → Add-on options → tbkeys-lite

Copy/Paste to Settings


{
    "j": "cmd:cmd_nextMsg",
    "k": "cmd:cmd_previousMsg",
    "o": "cmd:cmd_openMessage",
    "f": "cmd:cmd_forward",
    "d": "cmd:cmd_delete",
    "r": "cmd:cmd_reply",
    "a": "cmd:cmd_replyall",
    "x": "cmd:cmd_archive",
    "~": "cmd:cmd_goStartPage",
    "/": "cmd:cmd_search",
    "c": "func:MsgNewMessage",
    "u": "tbkeys:closeMessageAndRefresh"
}
