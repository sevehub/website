Remarkpy Theme - AsciiMath - PyScript

AsciiMath is an easy-to-write markup language for mathematics. It's easier than Latex and it's supported by MathJax display engine. Unfortunately, markdown to HTML renders often uses asciimath delimiters `...` for code blocks.

Remarkpy has a dedicated theme that replaces the default delimiters with a !!!.

The same theme includes also the PyScript library which allows running and displaying the output of python scripts directly inside a presentation, as shown in the figure below.
